# 使用参数

```shell
参数
  -gas uint 增加的 gas limit
        -gas 10 (default 10)
  -key string   bundler 密钥
        -key 
  -origin string
        -origin 
  -price uint 增加的 gas price
        -price 10 (default 10)
  -ws string  ws 地址
        -ws wss://localhost:8080 (default "wss://localhost:8080")
```
